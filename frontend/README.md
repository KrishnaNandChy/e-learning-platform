# E-Learning Platform Frontend

This is a modern, responsive React application for an E-Learning platform, built with Vite and Tailwind CSS.

## Features

- **Modern UI/UX**: Clean design inspired by top platforms.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Authentication**: Login and Signup with form validation.
- **Course Discovery**: 
  - Featured courses on Home page.
  - Course listing with categories and filtering.
  - Detailed course pages with curriculum and instructor info.
- **Dashboards**:
  - Student Dashboard for tracking progress.
  - Instructor Dashboard for managing courses.

## Tech Stack

- **React**: UI Library
- **Vite**: Build tool
- **Tailwind CSS**: Styling
- **Lucide React**: Icons
- **React Router DOM**: Navigation

## Project Structure

- `src/components`: Reusable UI components (Navbar, Footer, CourseCard).
- `src/pages`: Main page views (Home, Login, Courses, etc.).
- `src/context`: State management (Auth).
- `src/services`: API integration.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
